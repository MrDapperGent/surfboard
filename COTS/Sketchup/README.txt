Installing STL support
======================

1. Downloading extension from: https://extensions.sketchup.com/es/content/sketchup-stl

2. In SketchUp, select Window > Preferences (Microsoft Windows) or SketchUp > Preferences (Mac OS X).

3. In the dialog box that appears, select Extensions in the sidebar on the left.

4. On the Extensions pane, click the Install Extension button.

5. In the Open dialog box that appears, navigate to the .rbz file saved to your computer, select the file, 
   and click OK (Microsoft Windows) or Open (Mac OS X).

6. When SketchUp warns you about installing only extensions from trusted sources, 
   click Yes to proceed and install your extension. (This assumes you do, in fact, trust the source of your extension.)
